# 课程代码 Markdown 说明索引

每个代码文件旁边都有一个同名 Markdown，用于查看脚本目标、结构、运行方式和参考实现。

使用方式：

1. 上课先打开对应 `.md`。
2. 查看脚本目标、代码结构和运行方式。
3. 再打开同名代码文件，对照完整参考实现。
4. 最后按“可修改点”做小练习。
