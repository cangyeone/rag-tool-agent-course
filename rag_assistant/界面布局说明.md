## 📐 布局详解

### 左侧区域（宽度占比 25%）

**功能**：多对话管理

**组件**：
- **标题**："💬 多个对话"
- **新建按钮**："➕ 新建对话"（主要按钮）
- **对话列表**：
  - 每个对话显示为可点击的按钮
  - 当前选中对话高亮显示
  - 右侧有删除按钮（🗑️）
- **统计信息**：底部显示"共 N 个对话"

**交互**：
- 点击对话 → 切换到该对话
- 点击新建 → 创建空对话并切换
- 点击删除 → 删除对话并切换到下一个

### 右侧上方区域（高度占比 15%）

**功能**：配置面板

**布局**：两列等宽

**左列 - 知识库来源**：
- 标题："知识库来源"
- 内容：显示当前使用的知识库名称和路径
- 样式：信息框（蓝色背景）

**右列 - 检索设置**：
- 标题："检索设置"
- 滑块 1：召回片段数（2-8，默认 5）
- 滑块 2：向量权重（0.0-1.0，默认 0.75）
- 提示：鼠标悬停显示帮助文本

### 右侧中间区域（高度占比 50%）

**功能**：对话历史展示

**组件**：
- **容器**：固定高度 400px，带边框，可滚动
- **空状态**：无消息时显示欢迎提示
- **消息气泡**：
  - 用户消息：蓝色背景，左侧蓝色边框，"👤 您："前缀
  - AI 消息：灰色背景，左侧绿色边框，"🤖 AI："前缀
  - 支持 Markdown 渲染

**特性**：
- 自动滚动到底部
- 保留完整对话历史
- 刷新页面不丢失

### 右侧下方区域（高度占比 35%）

**功能**：提问输入区

**组件**：
- **分隔线**：视觉上区分对话历史和输入区
- **输入框**：
  - 占宽度 83%（5/6）
  - 占位符："输入您的问题..."
  - 支持回车发送
- **发送按钮**：
  - 占宽度 17%（1/6）
  - 文字："📤 发送"
  - 样式：主要按钮（蓝色）
- **示例问题**：
  - 折叠面板，默认收起
  - 展开后显示 4 个示例问题
  - 点击可快速填充到输入框

## 🎯 设计理念

### 1. 清晰的视觉层次
- 使用颜色区分不同功能区域
- 通过边框和阴影增强立体感
- 合理的留白避免拥挤

### 2. 高效的操作流程
- 常用操作（新建、发送）使用主要按钮
- 危险操作（删除）使用图标按钮
- 快捷提示减少输入成本

### 3. 友好的反馈机制
- 加载状态显示 spinner
- 错误消息使用红色警告框
- 成功操作即时刷新界面

### 4. 响应式设计
- 适配不同屏幕尺寸
- 侧边栏可折叠（移动端）
- 内容区域自适应高度

## 🔄 与旧版对比

| 维度 | 旧版 | 新版 |
|------|------|------|
| 布局结构 | 单栏 + 侧边栏 | 左右分栏 |
| 会话管理 | 无 | 多会话 + 持久化 |
| 配置位置 | 侧边栏 | 顶部双列 |
| 对话展示 | 每次覆盖 | 历史累积 |
| 输入方式 | text_area + button | text_input + button |
| 视觉风格 | 简洁 | 现代化气泡 |

##  下一步优化建议

1. **搜索功能**：在对话列表中增加搜索框
2. **导出功能**：支持导出对话为 Markdown/PDF
3. **标签系统**：为对话添加标签便于分类
4. **快捷键**：Ctrl+N 新建，Enter 发送，Ctrl+D 删除
5. **主题切换**：支持深色/浅色主题
6. **语音输入**：集成语音转文字功能
