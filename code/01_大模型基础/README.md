# 01 大模型基础

## 主要讲什么

- 这一章讲 AI 模型从传统机器学习、深度学习到大语言模型的基本脉络。重点不是追求模型训练细节，而是看懂文本如何进入模型、模型如何逐步生成输出，以及 chat 格式为什么会成为主流接口。

## 基本概念

- Token：模型处理文本时使用的基本单位。
- Embedding：把 token 转成向量表示，便于计算。
- Transformer：当前大语言模型的核心结构。
- Attention：让模型在生成时关注输入中更相关的部分。
- Chat messages：用 system、user、assistant 组织多轮对话。
- Stream：边生成边返回，降低等待感。
