## 讲解主线

先讲清"为什么需要工具"（模型自身的局限），再讲 Function Calling 协议和工具返回结果处理，然后深入到工具的工程化管理（schema、注册表、执行器、测试），最后进入 Agent 的高级话题（ReAct、Planner、多轮记忆、安全护栏）。

## 推荐运行顺序

```text
code/lesson_01_RAG加工具调用/
├── 02_工具调用的基本流程.py        → 调用链路：请求→识别→执行→返回
├── 04_工具返回结果处理.py          → 工具返回值解析与二次调用
└── 05_RAG_加工具端到端.py          → RAG + 工具的完整端到端演示

code/lesson_02_工具注册表与参数规范/

├── 03_工具执行器.py                 → 统一执行器，路由到具体函数
├── 04_工具错误处理.py               → 异常分类、重试、降级
└── 05_工具测试集.py                 → 批量工具测试，覆盖率统计
```

## 运行提示

先运行 lesson_01 建立工具调用基本认知，再运行 lesson_02 理解工具工程化管理，最后运行 lesson_03 进入 Agent 编排。lesson_03 的 01_ReAct 轨迹是理解 Agent 的核心脚本，建议重点讲解。
