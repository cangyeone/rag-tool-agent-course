# 09 专题：LLM 狼人杀

这个专题是一个独立的课堂案例，用“狼人杀”把大模型应用里的几个核心概念串起来：

- 角色扮演（Role Playing）
- 多智能体交互（Multi-Agent）
- 记忆管理（Memory）
- 推理与博弈（Reasoning）
- 结构化输出（JSON）
- 状态机（Game State）
- 工具调用（Tool Call）
- 裁判 Agent

## 运行前准备

脚本会读取系统环境变量 `DEEPSEEK_API_KEY`。公开仓库不提供任何真实 Key。

如果需要临时换成别的 Key，可以直接修改：

```text
code/.env
```

也可以用环境变量覆盖 `.env`。

进入目录：

```bash
cd rag-tool-agent-course/code/09_专题_多角色推理游戏/code
```

运行：

```bash
python 01_调API_最小chat.py
```

## 教学提示

这组代码故意保持简单，适合初学者逐行阅读。课堂上可以从以下位置修改：

- 修改玩家人数和身份。
- 修改每个角色的发言风格。
- 修改 `temperature`，观察模型发言稳定性变化。
- 修改投票数据，观察工具调用和裁判 Agent 的结果。
- 修改 memory 内容，观察多轮对话是否更连贯。
- 运行完整游戏脚本，观察系统如何循环推进并输出获胜阵营。
- 运行女巫版本，观察一次性资源如何进入 game state。
