## 8. 完整参考代码

```python
"""02 最小 next-token 循环。

不用真实模型，先用一个小词表说明“每次选下一个 token”。
"""

print("02 最小 next-token 循环")
print("=" * 72)

vocab = ["可以", "尝试", "候补申请", "，", "但", "不能", "保证", "成功", "。"]
# 每个 token 后面最可能接什么。真实模型会用神经网络计算分数，这里用字典类比。
next_map = {
    "可以": "尝试",
    "尝试": "候补申请",
    "候补申请": "，",
    "，": "但",
    "但": "不能",
    "不能": "保证",
    "保证": "成功",
    "成功": "。",
}

current = "可以"
result = [current]
print("起始 token：", current)

for step in range(8):
    next_token = next_map.get(current, "。")
    print(f"第 {step + 1} 步：{current!r} -> {next_token!r}")
    result.append(next_token)
    current = next_token
    if current == "。":
        break

print("\n拼接后的文本：", "".join(result))
print("要点：大模型生成长回答，本质上也是重复预测下一个 token。")
```
