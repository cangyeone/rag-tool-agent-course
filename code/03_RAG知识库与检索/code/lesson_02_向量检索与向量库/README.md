## 本节课目标

本节课按 5 个 demo 展开。每个 demo 都能单独运行，适合先讲 3-5 分钟，再可改一个参数观察输出。

## 演示脚本

### Demo 1: 文本向量化

脚本：`01_文本向量化.py`

演示重点：用稳定 hash 向量解释 embedding 的相似度思想。

运行方式：

```bash
python 01_文本向量化.py
```

### Demo 2: 余弦相似度

脚本：`02_余弦相似度.py`

演示重点：手写 cosine similarity，观察问题和资料的距离。

运行方式：

```bash
python 02_余弦相似度.py
```

### Demo 3: 向量库写入

脚本：`03_向量库写入.py`

演示重点：把向量和元数据保存成相对路径 JSON 文件。

运行方式：

```bash
python 03_向量库写入.py
```

### Demo 4: 向量库读取检索

脚本：`04_向量库读取检索.py`

演示重点：从本地 JSON 向量库读取并检索 top_k。

运行方式：

```bash
python 04_向量库读取检索.py
```

### Demo 5: 召回质量检查

脚本：`05_召回质量检查.py`

演示重点：检查检索结果是否包含候补申请、退款、学生优惠等目标关键词。

运行方式：

```bash
python 05_召回质量检查.py
```
