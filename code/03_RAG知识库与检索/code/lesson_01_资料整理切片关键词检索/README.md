## 本节课目标

本节课按 6 个 demo 展开。每个 demo 都能单独运行，适合先讲 3-5 分钟，再可改一个参数观察输出。

## 演示脚本

### 补充讲义: BM25 详细解释

文档：`BM25_详细解释.md`

用途：系统解释 BM25 的直觉、公式、参数、适用场景和与向量检索的关系，可配合 `04_关键词检索.py` 和 `05_检索结果解释.py` 讲解。

### Demo 1: 资料清单盘点

脚本：`01_资料清单盘点.py`

演示重点：把规则资料整理成 title、source、content 三个字段。

运行方式：

```bash
python 01_资料清单盘点.py
```

### Demo 2: PDF 文档解析

脚本：`02_PDF文档解析.py`

演示重点：把 PDF 按页解析成可入库文本，并保留来源、页码和标题。

输出文件：

```text
knowledge_base/parsed_pdf_pages.json
knowledge_base/parsed_pdf_text.txt
```

运行方式：

```bash
python 02_PDF文档解析.py
```

### Demo 3: 文档切片

脚本：`02_文档切片.py`

演示重点：演示 chunk_size 和 overlap 如何影响检索粒度。

运行方式：

```bash
python 02_文档切片.py
```

### Demo 4: overlap 作用

脚本：`03_overlap_作用.py`

演示重点：说明重叠窗口如何减少句子被截断造成的信息丢失。

运行方式：

```bash
python 03_overlap_作用.py
```

### Demo 5: 关键词检索

脚本：`04_关键词检索.py`

演示重点：用 Counter 做一个轻量关键词检索器。

运行方式：

```bash
python 04_关键词检索.py
```

### Demo 6: 检索结果解释

脚本：`05_检索结果解释.py`

演示重点：把命中片段、分数和来源打印给参与者看。

运行方式：

```bash
python 05_检索结果解释.py
```
