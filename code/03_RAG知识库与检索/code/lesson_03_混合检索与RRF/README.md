## 本节课目标

本节课按 5 个 demo 展开。每个 demo 都能单独运行，适合先讲 3-5 分钟，再可改一个参数观察输出。

## 演示脚本

### Demo 1: 关键词加向量混合检索

脚本：`01_关键词加向量混合检索.py`

演示重点：分别跑关键词和向量检索，再比较结果。

运行方式：

```bash
python 01_关键词加向量混合检索.py
```

### Demo 2: RRF 排名融合

脚本：`02_RRF_排名融合.py`

演示重点：用 Reciprocal Rank Fusion 融合两个检索列表。

运行方式：

```bash
python 02_RRF_排名融合.py
```

### Demo 3: 工具函数最小版

脚本：`03_工具函数最小版.py`

演示重点：写服务点代码、服务编号查询、退款提示三个确定性工具。

运行方式：

```bash
python 03_工具函数最小版.py
```

### Demo 4: RAG 加工具回答

脚本：`04_RAG_加工具回答.py`

演示重点：把规则资料和工具结果一起交给回答模板。

运行方式：

```bash
python 04_RAG_加工具回答.py
```

### Demo 5: 回答质检

脚本：`05_回答质检.py`

演示重点：检查最终回答是否包含依据、下一步和业务边界。

运行方式：

```bash
python 05_回答质检.py
```

### Demo 6: BGE-m3 真实向量库写入

脚本：`06_BGE-m3真实向量库写入.py`

演示重点：使用本地 `open_models/bge-m3` 生成真实 embedding，写入 FAISS 索引和 JSONL 元数据。

运行方式：

```bash
python 06_BGE-m3真实向量库写入.py
```

### Demo 7: BGE-m3 真实向量检索与 RRF

脚本：`07_BGE-m3真实向量检索与RRF.py`

演示重点：对比 BM25 关键词检索、BGE-m3 向量检索和 RRF 排名融合。

运行方式：

```bash
python 07_BGE-m3真实向量检索与RRF.py
```
