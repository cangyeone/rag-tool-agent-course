## 讲解主线

从上下文窗口管理切入，逐步深入到记忆系统设计、多工具协同、上下文压缩与摘要、Agent 自我修正，最后总结上下文工程最佳实践。顶层两个综合脚本（04_多工具Agent.py、05_工作流状态机.py）将前面的知识串联起来。

## 推荐运行顺序

```text
code/
├── 04_多工具Agent.py                → 6 步流水线，工具编排综合演示
└── 05_工作流状态机.py               → 基于状态的工作流，含质量检查节点

code/lesson_01_多工具Agent与可控执行/
├── 01_上下文窗口管理.py             → token 预算、窗口滑动、截断策略
├── 02_记忆系统设计.py               → 短期/长期记忆架构，检索式记忆
├── 03_多工具协同.py                 → 多工具并发调用与结果合并
├── 04_上下文压缩与摘要.py           → 长对话压缩策略，摘要生成
└── 05_Agent自我修正.py              → 执行结果校验与自动修正循环
```

## 运行提示

建议先运行 lesson_01 逐文件建立上下文工程认知，再运行顶层的 `04_多工具Agent.py` 和 `05_工作流状态机.py` 看综合效果。上下文窗口管理（01）和记忆系统设计（02）是本课核心，建议重点讲解。
