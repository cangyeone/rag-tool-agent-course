## 学习方式

每个 `.py` 文件旁边都有同名 `.py.md`。建议这样学习：

1. 先读 `.py.md`，理解目标、输入、输出、步骤。
2. 让 OpenCode 根据 `.py.md` 解释现有脚本。
3. 让 OpenCode 按 `.py.md` 重新实现一个更简单版本。
4. 运行脚本，记录输出。
5. 修改问题、参数或工具描述，再运行一次。

## 推荐 OpenCode 指令

```text
请阅读这个 Markdown 任务书，然后检查同名 Python 脚本是否完整实现了要求。
请把脚本改得更适合初学者阅读：少用函数，按步骤顺序写，保留详细注释。
请增加一个可修改参数，并在 Markdown 中说明如何修改。
请运行脚本并根据报错修复，但不要写入任何 API Key。
```
